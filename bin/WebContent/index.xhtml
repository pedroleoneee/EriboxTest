Eribox Test

INSTRUÇÕES GERAIS

  • Criar um CRUD de 'Produto' usando as classes, páginas e configurações já existentes na aquitetura do projeto.
  • A arquitetura do projeto foi desenvolvida utilizando JSF (Primefaces), Spring Framework e JPA (Hibernate provider) e Tomcat 9.
  • A estutura base que deverá ser usada para a construção do projeto está no gihub no repositório https://github.com/cristripoli/EriboxTest.
  • Caso você sinta necessidade, novas classes, configurações e páginas podem ser adicionadas, porém as já existentes devem ser utilizadas.
  • A aplicação está configurada para usar o banco MySQL 5.1 utilzando o esquema eribox_test para acesso localhost.
  • O produto deve ter ao menos informações de nome, código e preço.
  • Validações, coesão, acoplamento, estética e qualidade do código serão levados em consideração na avaliação.

INSTRUÇÕES DE ENVIO

  • Para envio do projeto você de subir o projeto base para um repositório na sua conta do Github.
  • O repositório de envio deve ser privado.
  • Após finalizado a projeto envie o link do seu repositório por email e adicione o usuário cristripoli como colaborador.
